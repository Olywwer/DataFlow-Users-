# DataFlow Users - CRUD com JSON Server e Express

Sistema completo de CRUD (Create, Read, Update, Delete) com busca por CPF, utilizando JSON Server como banco de dados e Node.js com Express.

## 🚀 Tecnologias

- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: Node.js, Express
- **Database**: JSON Server
- **Hospedagem**: Render

## 📋 Funcionalidades

- ✅ Cadastrar usuário (POST)
- ✅ Listar usuários (GET)
- ✅ Atualizar usuário (PUT)
- ✅ Excluir usuário (DELETE)
- ✅ Buscar por CPF
- ✅ Design responsivo
- ✅ Tema claro/escuro
- ✅ Dashboard com gráficos
- ✅ Exportar dados (JSON)
- ✅ Gerar relatório PDF
- ✅ Filtros avançados
- ✅ Ordenação na tabela
- ✅ Paginação
