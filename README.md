DataFlow-Users
CRUD com JSON Server e Express


Sistema completo de CRUD (Create, Read, Update, Delete) com busca por CPF, utilizando JSON Server como banco de dados e Node.js com Express para servir as páginas HTML.

Sobre o Projeto

Este projeto é um sistema de gerenciamento de usuários que implementa todas as operações de um CRUD:
- POST - Cadastro de novos usuários
- GET - Listagem de todos os usuários
- PUT - Atualização de dados dos usuários
- DELETE - Remoção de usuários

Funcionalidade Especial: Busca por CPF

A função `buscarDados()` permite localizar um usuário utilizando o CPF como chave de busca, ao invés do ID tradicional. Isso proporciona uma experiência mais natural para o usuário final, que geralmente conhece o CPF do cliente.

Tecnologias Utilizadas

- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: Node.js, Express
- **Database**: JSON Server
- **Hospedagem**: Render
- **Versionamento**: Git/GitHub
